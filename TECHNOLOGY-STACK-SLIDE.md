## 🎯 ДЕТАЛЬНАЯ СТРУКТУРА:

### ГРУППА 1: ОРКЕСТРАЦИЯ

**Docker Swarm**
- Встроенный оркестратор Docker
- Управление жизненным циклом сервисов
- Автоматическое восстановление и балансировка нагрузки
- Простота настройки для учебного проекта

---

### ГРУППА 2: БЭКЕНД И ФРОНТЕНД

**Фронтенд:**
- **HTML** - структура веб-страниц
- **CSS** - стилизация интерфейса
- **JavaScript** (Vanilla JS) - интерактивность и взаимодействие с API

**Бэкенд:**
- **FastAPI** - высокопроизводительный Python-фреймворк для REST API
- **PostgreSQL** - транзакционная СУБД для хранения данных
- **Python 3.11** - язык программирования
- **Nginx** - веб-сервер для статических файлов фронтенда

**Дополнительные библиотеки:**
- SQLAlchemy (ORM)
- PyJWT (JWT-аутентификация)
- bcrypt (хеширование паролей)
- ReportLab (генерация PDF)

---

### ГРУППА 3: КОНТЕЙНЕРИЗАЦИЯ

**Docker**
- Контейнеризация всех компонентов системы
- Изоляция окружений
- Переносимость приложений
- Docker Compose для локальной разработки

## 📝 ДЕТАЛЬНЫЙ СПИСОК ТЕХНОЛОГИЙ:

### Оркестрация:
- **Docker Swarm** - оркестратор контейнеров

### Контейнеризация:
- **Docker** - платформа контейнеризации
- **Docker Compose** - для локальной разработки

### Фронтенд:
- **HTML** - разметка
- **CSS** - стили
- **JavaScript** (Vanilla JS) - логика клиента
- **Nginx** - веб-сервер для статики

### Бэкенд:
- **Python 3.11** - язык программирования
- **FastAPI** - веб-фреймворк
- **PostgreSQL** - база данных
- **SQLAlchemy** - ORM
- **PyJWT** - JWT-токены
- **bcrypt** - хеширование паролей
- **ReportLab** - генерация PDF
- **HTTPX** - HTTP-клиент

### DevOps/CI/CD:
- **GitHub Actions** - CI/CD-конвейер
- **Docker Hub** - реестр образов
